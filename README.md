# wp-bootstrap-blog
WordPress Bootstrap theme starter
# Hướng dẫn cài đặt

1. Mở giao diện dòng lệnh -> Chạy `'bower install'` để cập nhật bootstrap
2. Thực hiện cài đặt giao diện
3. Xem kết quả
